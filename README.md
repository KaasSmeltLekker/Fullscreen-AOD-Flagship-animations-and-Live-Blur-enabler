# Fullscreen-AOD-Flagship-animations-and-Live-Blur-enabler

Enables Fullscreen AOD, Flagship animations, Live Blur and partial Galaxy AI features.

Designed for OneUI 6 and later versions.

To use it, just flash the module via Magisk/KernelSU/SukiSU/APatch.
